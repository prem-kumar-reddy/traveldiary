Title - TravelDiary

How to run the application:
1. Extract the .zip file
2. Navigate to the traveldiary folder.
3. In the terminal run "npm install".
4. Then run "nodemon server.js".
4. Open another terminal and Navigate to client folder.
5. Run "npm install".
6. Then "npm start".

After sometime the server starts on localhost:5000 and Client starts on localhost:3000
